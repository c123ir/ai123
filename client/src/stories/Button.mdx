# دکمه (Button)

دکمه‌ها به کاربران اجازه می‌دهند تا با یک کلیک، عملیات مشخصی را انجام دهند. آن‌ها برای شروع یک اقدام، مانند "ارسال یک فرم"، "ثبت تغییرات"، یا "تأیید یک انتخاب" استفاده می‌شوند.

## نمونه پایه

دکمه پایه با چهار نوع اصلی: اولیه (primary)، ثانویه (default)، متنی (text) و لینکی (link).

<Canvas of={ButtonStories.Primary} />

## انواع دکمه

- **اولیه (Primary)**: برای اقدامات اصلی و مهم
- **ثانویه (Default)**: برای اقدامات ثانویه و کمتر برجسته
- **خطرناک (Danger)**: برای اقدامات خطرناک یا برگشت‌ناپذیر
- **متنی (Text)**: برای اقدامات با اهمیت کمتر
- **لینکی (Link)**: برای هدایت کاربر به صفحات دیگر

## نکات استفاده

- از دکمه **اولیه (Primary)** فقط برای اقدام اصلی در هر صفحه یا بخش استفاده کنید.
- برای اقدامات خطرناک، همیشه از دکمه **خطرناک (Danger)** استفاده کنید و در صورت امکان یک تأییدیه از کاربر بگیرید.
- دکمه‌ها باید دارای برچسب‌های کوتاه و واضح باشند که عمل را توضیح می‌دهند (مثلاً "ذخیره" به جای "کلیک کنید").
- از دکمه‌های **غیرفعال (Disabled)** فقط زمانی استفاده کنید که می‌خواهید به کاربر نشان دهید این عمل در حال حاضر در دسترس نیست.
- برای دکمه‌های مرتبط با فرم، مانند "ارسال" یا "ذخیره"، از ویژگی `htmlType="submit"` استفاده کنید.

## پیاده‌سازی

### وارد کردن کامپوننت

```jsx
import { Button } from 'antd';
```

### استفاده پایه

```jsx
// دکمه اولیه
<Button type="primary">دکمه اولیه</Button>

// دکمه ثانویه
<Button>دکمه ثانویه</Button>

// دکمه خطرناک
<Button type="primary" danger>حذف</Button>

// دکمه بارگذاری
<Button type="primary" loading>در حال بارگذاری</Button>

// دکمه آیکون‌دار
<Button type="primary" icon={<SearchOutlined />}>جستجو</Button>

// دکمه غیرفعال
<Button type="primary" disabled>غیرفعال</Button>

// دکمه بلاک
<Button type="primary" block>دکمه تمام عرض</Button>
```

## دسترسی‌پذیری

- دکمه‌ها باید برچسب‌های توصیفی داشته باشند که عملکرد آن‌ها را به وضوح نشان دهد.
- برای دکمه‌های آیکون بدون متن، همیشه یک `aria-label` یا `title` اضافه کنید.
- از تباین رنگی کافی بین دکمه و متن آن اطمینان حاصل کنید.
- دکمه‌ها باید با کیبورد قابل دسترسی باشند (کلید Tab و Enter).

```jsx
// دکمه آیکون با دسترسی‌پذیری مناسب
<Button icon={<DeleteOutlined />} aria-label="حذف آیتم" title="حذف آیتم" />
```
