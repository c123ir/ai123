# کتابخانه کامپوننت‌های پارسی

به کتابخانه کامپوننت‌های پارسی خوش آمدید. این مجموعه شامل کامپوننت‌های متنوعی است که با تمرکز بر زبان فارسی و قالب راست به چپ (RTL) طراحی شده‌اند.

## معرفی

این کتابخانه بر پایه Ant Design ساخته شده و با فونت «وزیر» بهینه شده است تا تجربه کاربری مناسبی برای کاربران فارسی‌زبان فراهم کند.
همه کامپوننت‌ها از راست به چپ پشتیبانی می‌کنند و متناسب با نیازهای پروژه‌های فارسی تنظیم شده‌اند.

## ویژگی‌ها

- **پشتیبانی کامل از RTL**: تمام کامپوننت‌ها به صورت راست به چپ نمایش داده می‌شوند.
- **فونت وزیر**: استفاده از فونت استاندارد و خوانای وزیر برای نمایش متون فارسی.
- **سازگاری با اعداد فارسی**: نمایش صحیح اعداد فارسی در تمامی کامپوننت‌ها.
- **طراحی مدرن**: استفاده از طراحی مدرن و زیبا مطابق با استانداردهای روز.

## نحوه استفاده

برای استفاده از این کامپوننت‌ها، کافیست آن‌ها را مستقیماً از مسیر اصلی پروژه وارد (import) کنید.

```jsx
import { Button, Input, Table } from '../components';

function MyComponent() {
  return (
    <div>
      <h1>فرم نمونه</h1>
      <Input placeholder="نام خود را وارد کنید" />
      <Button type="primary">ذخیره</Button>
    </div>
  );
}
```

## پیکربندی تم

تمام کامپوننت‌ها قابلیت سفارشی‌سازی دارند و می‌توانید رنگ‌ها و سایر ویژگی‌های ظاهری آن‌ها را تغییر دهید.

```jsx
import { ConfigProvider } from 'antd';

// تعریف تم سفارشی
const theme = {
  token: {
    colorPrimary: '#3498db',
    borderRadius: 6,
    fontFamily: 'Vazir, sans-serif',
  },
};

// اعمال تم به کل برنامه
function App() {
  return (
    <ConfigProvider theme={theme} direction="rtl">
      <YourApp />
    </ConfigProvider>
  );
}
```
